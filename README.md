# day-15

# Data Quality Problems
Poor data quality includes missing, incorrect, or inconsistent information. It can lead to errors and poor decision-making.

# Migration Discussion
Data migration is the process of transferring data from one system to another. Proper planning ensures accuracy and minimizes data loss.

# Duplicate Prevention Ideas
Duplicates can be prevented using validation rules, matching criteria, and unique identifiers. Regular data cleaning also helps maintain accuracy.

# Enterprise Risks of Bad Data
Bad data can cause reporting errors, customer dissatisfaction, and operational inefficiencies. It may also lead to financial and compliance risks.

# Reflection
Maintaining high-quality data is essential for business success. Good data management improves accuracy, efficiency, and decision-making.
